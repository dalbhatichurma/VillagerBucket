# VillagerBucket Plugin

Capture individual villagers in buckets and place them anywhere!

## Features
- Capture villagers by right-clicking with a special bucket
- Preserves villager profession, level, type, and custom name
- Place villagers by right-clicking the ground
- Simple and lightweight

## Commands
- `/villagerbucket` or `/vb` - Get a villager bucket

## Permissions
- `villagerbucket.give` - Get villager buckets (default: op)
- `villagerbucket.capture` - Capture villagers (default: true)
- `villagerbucket.place` - Place villagers (default: true)

## Usage
1. Get a villager bucket with `/vb`
2. Right-click any villager to capture it
3. Right-click the ground to place the villager

## Installation
1. Drop `VillagerBucket.jar` into your plugins folder
2. Restart your server
3. Configure in `plugins/VillagerBucket/config.yml`

## Requirements
- Minecraft 1.21+
- Java 21+
